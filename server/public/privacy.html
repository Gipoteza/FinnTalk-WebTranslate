Политика конфиденциальности

Последнее обновление: апрель 2025

1. Общие положения

Расширение FinnTalk («Расширение») предназначено для автоматического перевода финских веб-страниц на русский язык. Настоящая политика описывает, какие данные обрабатываются при использовании Расширения.

2. Какие данные обрабатываются

Текст веб-страниц — текстовое содержимое страниц, которые пользователь выбирает для перевода. Текст передаётся на прокси-сервер для обработки через OpenAI API.

Расширение не собирает и не хранит:

  • Личные данные пользователя (имя, email, адрес)
  • Историю посещённых страниц
  • Данные форм или паролей
  • Финансовую информацию

3. Как используются данные

Текст страницы передаётся исключительно для выполнения перевода:

  • Текст отправляется на прокси-сервер finntalk-webtranslate-production.up.railway.app
  • Прокси-сервер передаёт текст в OpenAI API для перевода
  • Переведённый текст возвращается в браузер и отображается на странице
  • Текст не сохраняется на серверах после обработки запроса

4. Разрешения расширения

activeTab — используется для чтения текста активной вкладки при нажатии кнопки «Перевести страницу». Расширение не имеет доступа к другим вкладкам.

storage — используется для сохранения настроек расширения (выбранная модель, лимиты запросов) локально в браузере пользователя. Данные не передаются третьим лицам.

Доступ к хосту — расширение обращается только к finntalk-webtranslate-production.up.railway.app для выполнения перевода.

5. Удалённый код

Расширение не загружает и не выполняет удалённый код. Все скрипты расширения включены в ZIP-пакет. Обращения к внешнему серверу — только API-запросы для перевода текста.

6. Передача данных третьим лицам

Текст страниц передаётся в OpenAI для выполнения перевода в соответствии с политикой конфиденциальности OpenAI. Данные не передаются другим третьим лицам.

7. Безопасность

API-ключ OpenAI хранится исключительно в переменных окружения сервера Railway и никогда не передаётся в браузер пользователя.
